# SD 1.5 / SD 2.x

Stable Diffusion 1.5 and 2.x checkpoints — single-file format, widely supported. SD 1.5 has the largest model library on Civitai; SD 2.x is less common but improves on composition.

## Files needed in ComfyUI

| Field | File | ComfyUI folder |
|---|---|---|
| Checkpoint | e.g. `v1-5-pruned-emaonly.safetensors` | `models/checkpoints/` |
| VAE *(optional)* | e.g. `vae-ft-mse-840000-ema-pruned.safetensors` | `models/vae/` |
| IPAdapter model *(optional, adapter mode only)* | e.g. `ip-adapter_sd15.safetensors` | `models/ipadapter/` |

The VAE baked into most community checkpoints is fine. An external VAE only helps if colours look washed out.

## Where to download

- **SD 1.5 checkpoints** — [Civitai](https://civitai.com) (filter by SD 1.5), or [Hugging Face runwayml/stable-diffusion-v1-5](https://huggingface.co/runwayml/stable-diffusion-v1-5)
- **SD 2.1** — [Hugging Face stabilityai/stable-diffusion-2-1](https://huggingface.co/stabilityai/stable-diffusion-2-1)
- **VAE** — [Hugging Face stabilityai/sd-vae-ft-mse-original](https://huggingface.co/stabilityai/sd-vae-ft-mse-original) (`vae-ft-mse-840000-ema-pruned.safetensors`)
- **IPAdapter models** — [Hugging Face h94/IP-Adapter](https://huggingface.co/h94/IP-Adapter) (`models/sd15/` folder). No separate CLIP Vision file needed — it is bundled automatically by the loader.

## Required custom nodes

- **IPAdapter** *(only if using adapter reference mode)* — [cubiq/ComfyUI_IPAdapter_plus](https://github.com/cubiq/ComfyUI_IPAdapter_plus)

## Notes

Negative prompts and CFG scale are most effective here. Typical CFG: 7. Typical steps: 20–30 with DPM++ 2M Karras sampler.
